# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What This Repo Is

This is a **static deployment repository** for a personal portfolio site. It contains only the compiled output — not the source code. The single `index.html` is a bundled artifact (~505KB) generated by an external build tool, containing:

- Inline gzip-compressed, base64-encoded asset payloads (JavaScript, fonts, PDF)
- A client-side decompression and loading system using `DecompressionStream` and `Blob` URLs
- Custom web components (`<x-dc>`, `<sc-if>`, `<sc-for>` directives)

The source project that generates this file lives elsewhere; this repo exists to host the built output (e.g., via GitHub Pages or similar static hosting).

## Working with the Site

There are no build scripts, package managers, or dev servers in this repo. To preview the site locally, open `index.html` directly in a browser or serve it with any static file server:

```sh
npx serve .
# or
python3 -m http.server 8080
```

## Making Changes

Because `index.html` is machine-generated, **do not hand-edit it** unless making minor text or style changes that are clearly isolated. Direct edits will be overwritten the next time the source project is rebuilt and the output is re-deployed here.

For structural or content changes, the source project (not in this repo) must be modified and rebuilt, then the new `index.html` committed here.

## Deployment

Commit the updated `index.html` to `main`. The remote is `git@github.com:yashmehta001/portfolio-site.git`. If GitHub Pages is configured, pushing to `main` deploys automatically.
